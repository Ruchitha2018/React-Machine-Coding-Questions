# React Accordion Component

A simple and reusable **Accordion Component** built using **React Hooks**.
This project demonstrates both **single open** and **multiple open** accordion behavior.

---

## 🚀 Features

* Toggle accordion items
* Supports **single accordion open**
* Supports **multiple accordion open**
* Clean state management using **React Hooks**
* Simple and readable component structure

---

## 🛠 Tech Stack

* React
* JavaScript (ES6+)
* HTML
* CSS

---

## 📂 Project Structure

```
src
 ├── components/AccordionComponent.js
 ├── App.js
 └── index.js
```

---

## 📸 Demo

Accordion with two modes:

* **Single Open Mode** – Only one accordion can open at a time.
* **Multiple Open Mode** – Multiple accordions can be expanded simultaneously.
